# Coffee-Sales-Report
1. Project Title / Headline
Coffee Shop Sales Dashboard | Power BI

2. Short Description
The Coffee Shop Sales Dashboard is an interactive Power BI project designed to analyze coffee shop sales performance. It provides insights into revenue, orders, product performance, store performance, and sales trends through interactive dashboards and KPIs. The dashboard helps management make data-driven decisions to improve sales and business growth.

3. Tech Stack
Power BI Desktop
Power Query
DAX (Data Analysis Expressions)
Excel
Data Modeling

4. Data Source
The dashboard is built using a coffee shop sales dataset containing daily transaction records.

Dataset includes:
Transaction ID
Transaction Date
Transaction Time
Store Location
Product Category
Product Type
Product Detail
Quantity Sold
Unit Price
Revenue

5. Features & Highlights
   
Executive Dashboard
Displays Total Revenue, Total Orders, Total Quantity Sold, Average Order Value, and Average Selling Price.
Provides an overview of overall business performance.

Product Performance
Analyze revenue and quantity sold by product category and product type.
Identify top-selling and low-performing products.

Store Performance
Compare revenue, orders, and quantity sold across different store locations.
View store rankings and contribution to total sales.
Time Analysis

Track monthly, daily, and hourly sales trends.
Compare weekday and weekend sales performance.
